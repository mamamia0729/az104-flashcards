# AZ-104 Flashcards

Free, open-source flashcards for the **Microsoft Azure Administrator (AZ-104)** exam.

Black & white. No distractions. Exam-ready wording.

**Live site:** [mamamia0729.github.io/az104-flashcards](https://mamamia0729.github.io/az104-flashcards/)

---

## What's inside

### Day 1 — Resource Groups, VNets, Subnets, NSGs & Bastion

20 flashcards covering foundational Azure networking and governance:

| Category | Topics |
|----------|--------|
| Governance | Resource Groups, naming conventions (CAF), tags, tag inheritance |
| Networking | Virtual Networks, CIDR notation, subnets, reserved IPs, reserved subnet names |
| Security | NSGs, rule evaluation & priority, Azure Bastion, RDP security |
| Cost Management | Budget alerts, alert behavior |

Every card is worded to match the language and depth tested on AZ-104.

---

## Features

- Click or press `Space` to flip cards
- `Arrow Left` / `Arrow Right` to navigate
- Swipe on mobile
- Filter by category (Governance, Networking, Security, Cost Mgmt)
- Shuffle and reset
- Progress bar
- Dark mode (follows system preference)
- Fully responsive — works on phone, tablet, desktop
- Single HTML file, zero dependencies
